# Serwer Bankowy

### Skład zespołu: Tomasz Mazewski.

### Instrukcja techniczna: 
Kompilacja serwera:

` javac Main.java`

Uruchamiamanie serwera: 

` java Main`

Kompilacja client:

` javac Client.java`

Uruchamiamanie serwera:

` java Client`

### Deklaracja użycia sztucznej inteligencji: 

Przy tworzenie aplikacji korzystano ze sztucznej inteligencji (ChatGPT - darmowa wersja) przy:
- poprawienie klasy Serwer i Client
- stworzenie testów
